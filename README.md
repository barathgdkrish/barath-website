## Environment Configuration

This project requires a `src/environments/environment.ts` file to store sensitive configuration like Firebase keys. This file is ignored by Git for security.

### Setting up `environment.ts`

Create the file at `src/environments/environment.ts` with the following structure:

```typescript

  production: false,
  firebase: {
    apiKey: "YOUR_API_KEY",
    authDomain: "YOUR_AUTH_DOMAIN",
    projectId: "YOUR_PROJECT_ID",
    storageBucket: "YOUR_STORAGE_BUCKET",
    messagingSenderId: "YOUR_SENDER_ID",
    appId: "YOUR_APP_ID",
    measurementId: "YOUR_MEASUREMENT_ID"
  },
  googlePicker: {
    clientId: "YOUR_CLIENT_ID",
    developerKey: "YOUR_DEVELOPER_KEY",
    appId: "YOUR_APP_ID"
  }
};
```
